What is Javascript?

Data Types

  • Boolean
    • True
    • False
  • Null
    • Is an intentional absence and treated as falsy.
  • Undefine
    • Variable has not been assigned a value and its treated as falsy.
  • Number

  • String
    • Anything in " " or ' ' is a sting.
    • If a number "3" is in quotes, it becomes a string.

Variables

Const

Variable identifier cannot be reassigned and value can't be changed through reassignment.
Unless its being used for an object/array the values are mutable.

It's mostly used for functions or for other values that won't

Let

Its value are mutable.

Meaning you can change its value.

  • let i = 0;
  • i++
  • i = 1

    • Could be used for something like a score counter for a game. Every time you get a point the value changes.

    Operators

      Arithmetic Operators

    • Addition (+)

      Ex: let add = 1 + 1;

      Result: add = 2

    • Subtraction (-)

      Ex: let sub = 1 - 1;

      Result: sub = 0

    • Multiplication (*)

      Ex: let multiply = 1 * 3;

      Result: multiply = 3

    • Exponent (**)

      Ex: let one = 5 ** 2;

      Result: one = 25

    • Division (/)

      Ex: let divide = 10 / 2;

      Result: divide = 5

    • Remainder (%)

      Ex: let remain = 10 % 3;

      Result: remain = 1

      Explanation:
      • 3 goes into 10 three times

      • 3 * 3 = 9

      • 10 - 9 = remainder of 1

    • Increment (++)

      Ex: let addOne = 1; addOne++

      Result: addOne = 2

    • Decrement (--)

      Ex: let minusOne = 2; minusOne--

      Result: minusOne = 1
